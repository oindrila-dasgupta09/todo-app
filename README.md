🚀 Todo App (Full Stack + DevOps)
📌 Project Overview
This is a Full Stack Todo Application built using React, Node.js, PostgreSQL, Docker, and Kubernetes.
The application allows users to:
•	Add tasks
•	Update tasks
•	Delete tasks
•	View tasks
The project was also containerized using Docker and deployed using Kubernetes to understand real-world DevOps workflows.
